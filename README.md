# Gurdwara Sahib Switzerland

Welcome to the official website of Gurdwara Sahib Switzerland!

## Project Overview
This website serves as a digital sanctuary and information hub for the Gurdwara Sahib community in Switzerland. It features:
- **Spiritual Content**: Daily Hukamnama and Diwan schedules.
- **Community Services**: Information on Langar, Sewa, and religious programs.
- **Education**: Details about the Gurmat School and annual youth camps.
- **Visitor Information**: Guidelines for visitors, contact details, and location map.

## Performance & Design
The site is built with **Next.js 15** and **Framer Motion**, focusing on:
- High performance and Vercel-ready architecture.
- Majestic, premium design with professional typography.
- Multi-language support (English, German, Punjabi).
- SEO-optimized metadata for all pages.
